# TV premiere schedule

Android-приложение для просмотра расписания премьер телевизионных шоу. Приложение позволяет пользователям выбирать страну и дату, чтобы получить актуальное расписание премьер по api TVMaze.

## Функции

- **Выбор страны и даты для запроса**: Выбор данных с помощью удобных виджетов
- **Просмотр расписания**: Детальная информация о премьерах
- **Локальное хранение**: Кеширование данных в локальной базе данных Room
- **Поддержка двух языков**: Интерфейс на русском и английском языках

## Архитектура

Приложение построено на принципах **Clean Architecture** с разделением на слои:

- **Presentation**: Jetpack Compose UI, ViewModel, Navigation
- **Domain**: Use Cases, Repository interfaces, бизнес-логика
- **Data**: Репозитории, API клиент, локальная база данных

Используется паттерн **MVVM** для разделения логики и UI.

## Основные технологии:
- **Язык**: Kotlin
- **Архитектура**: MVVM + Clean Architecture
- **UI Framework**: Jetpack Compose
- **DI**: Hilt (Dagger)
- **База данных**: Room
- **Сеть**: Retrofit + OkHttp
- **Асинхронность**: Kotlin Coroutines + Flow
- **Навигация**: Navigation Compose
- **Изображения**: Coil
- **Unit тесты**: JUnit + MockK
- **Интеграционные тесты**: Espresso
- **Compose тесты**: Compose UI Test
